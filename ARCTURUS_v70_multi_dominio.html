Explicabilidade · SHAP

Importância das Features

Contribuição de cada feature para as decisões do modelo (Gini importance). O modelo aprende padrões de escrita — estrutura, não tópico.

Como funciona

Metodologia

Pipeline transparente e reproduzível — do dado bruto ao score final com explicabilidade.

📡
Coleta de Dados
Artigos retratados coletados via OpenAlex API. 100 casos confirmados de paper mills + 397 controles legítimos pareados por área e período (2018–2024).
OpenAlex · Crossref
🧠
Feature Engineering
9 features textuais manuais + embeddings multilíngues (Sentence-Transformers). O papermill_index é uma combinação ponderada de tokens e frases suspeitas.
9 features · Embeddings
🗺️
UMAP + XGBoost
Redução de dimensionalidade com UMAP para visualização e clustering. Classificação com XGBoost (scale_pos_weight) + BorderlineSMOTE para balanceamento.
XGBoost · UMAP · SMOTE
📊
Explicabilidade SHAP
Cada predição acompanhada de explicação das top features via SHAP values. Transparência total: sabe-se exatamente por que um abstract foi sinalizado.
SHAP · Interpretabilidade
⚖️
Limitações Declaradas
497 amostras ainda é dataset modesto. Features textuais simples não capturam fraude sofisticada. Escopo atual: biomedicina em inglês. Falsos positivos esperados em triagem.
Triagem · Não diagnóstico
📖
Código Aberto
Pipeline completo disponível no GitHub para reprodução. Dataset baseado em abstracts públicos via OpenAlex. Preprint em preparação para OSF Preprints.
Open Source · OSF Preprint

Evolução

Roadmap

Do protótipo ao produto — cada versão com validação honesta e métricas reais.

v4.2 — v6.3
Prova de Conceito
Dataset inicial de 48 artigos → expansão para 285. Feature engineering, correção de bugs, pipeline de coleta via OpenAlex. Modelo Random Forest baseline.
v7.0
Validação Externa
Hold-out de 57 artigos separado antes do treino. Gap de 7.5 pp entre CV interno e externo. Generalização confirmada. Interface multi-domínio com 7 áreas.
v8.0 · Atual
XGBoost + UMAP + SHAP
AUC-ROC de 90.88%. Dataset expandido para 497 amostras. XGBoost substituindo Random Forest. Embeddings multilíngues com Sentence-Transformers. Explicabilidade SHAP.
v9.0 · Planejado
API Pública + OSF Preprint
Endpoint REST para submissão de DOIs. Dashboard com explicabilidade SHAP completa. Publicação no OSF Preprints. Análise de rede de coautoria.
v10.0 · Futuro
Produto SaaS
SaaS para editoras e instituições. Planos Starter e Pro. Auditoria institucional. Parceria estratégica com Retraction Watch.
